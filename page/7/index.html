分类: html

设置一个回到顶部的按钮

document.querySelector(".get-top").addEventListener("click", () => {
  window.scrollTo({
    top: 0,
    behavior: "smooth",
  });
});

window.addEventListener("scroll", () => {
  if (window.scrollY !== 0) {
    document.querySelector(".get-top").style.opacity = "1";
  } else {
    document.querySelector(".get-top").style.opacity = "0";
  }
});

关于GitHub403 timeout的设置
分类: 杂识

最近这段时间 Github 体验极差,不知道是我的问题还是怎样,每次推送文章\提交分支都在频繁 403,烦得我半夜猛砸键盘,以至于去厕所吃了坨屎才冷静下来.

使用代理软件的话,可以去看一下软件的代理端口,然后设置一下 git 的 http 代理

git config --global http.proxy 127.0.0.1:8087

取消代理

git config --global --unset http.proxy
